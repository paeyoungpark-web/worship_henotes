🎛 컨트롤 가이드

🎚 페이더 (FADER)

각 채널의 볼륨을 dB 단위로 조절합니다. 0 dB가 기본값이며, 위로 올리면 크게(+6 dB까지), 아래로 내리면 작게 들립니다.

🔇 MUTE (M)

해당 채널을 일시적으로 끕니다. 페이더 값은 유지되며, 해제 시 원래 볼륨으로 돌아옵니다.

🟡 SOLO (S)

이 채널만 듣고 나머지는 끕니다. 여러 채널을 SOLO로 켜면 그 채널들끼리만 들립니다.

🌫 REVERB (REV)

가상의 공간 울림을 추가합니다. 보컬 15~25%, 악기 5~15% 권장.

↔ PAN

좌(L) / 우(R) 스테레오 위치 조절. C(가운데)가 기본.

📍 SCENES (프리셋)

"MY CH"에서 자기 채널을 먼저 선택한 후 사용하세요.

🔁 LOOP A-B

SET A → SET B 순서로 구간 지정 후 LOOP ON 으로 무한 반복. 어려운 구간 연습에 유용.

📤 EXPORT MIX

현재 모든 채널의 볼륨/리버브/팬/뮤트 상태를 텍스트로 클립보드에 복사. 믹스 담당자에게 제안할 때 사용.
